How to use git commands

Step One!

The first step is to make your change to the files that you are working on. After you have made these changes you move onto step number 2

Step Twp!

In the next step you type into the git bash 'git branch name'. The name is what you want it to be. Then we move onto step 2

Step Three!

In this step you now check out to the branch you have created by typing into the bash shell 'git checkout name'. Next is step four.

Step Four

In order to commit these changes we must add them by typing in the following 'git add .'. Make sure to leave a space between add and the dot(.).

Step Five

In this last step you type in 'git commit -m ""'. In between the speech marks you can type in a message that you want to be read in the repository
